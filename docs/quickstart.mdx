## Prerequisites

<Info>
  You need a Microsoft Entra ID account with membership in at least one security group that maps to a LiteLLM team. Contact your administrator if you don't have access.
</Info>

## Get started

<Steps>
<Step title="Sign in and get provisioned">
  Navigate to the Apollos AI portal and click **Sign in with Microsoft**. You'll be redirected to Microsoft's login page for authentication.

  On first login, the portal automatically provisions your account:

  1. Validates your Entra ID credentials
  2. Resolves your security group memberships
  3. Creates your LiteLLM team memberships
  4. Generates an initial API key for each team

  A progress dialog shows each step as it completes.
</Step>

<Step title="Copy your API key">
  After provisioning completes, the portal displays your generated API key(s) with a **Copy** button. Copy each key before clicking **Continue to Portal** — they are shown only once.

  <Warning>
    Store your API key securely. You cannot retrieve it after this screen. If you lose it, go to **API Keys** and create a new one or rotate the existing key.
  </Warning>
</Step>

<Step title="Make your first request">
  Use your API key to send a request through the LiteLLM proxy:

  ```bash
  curl -X POST https://your-litellm-proxy/v1/chat/completions \
    -H "Authorization: Bearer sk-your-api-key" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "gpt-4o",
      "messages": [
        {"role": "user", "content": "Hello, world!"}
      ]
    }'
  ```

  The proxy routes your request to the appropriate LLM provider based on your team's model access configuration.

  <Tip>
    Go to **Models** in the sidebar and click any model card to see ready-to-use code examples in Python, curl, and TypeScript — tailored to that model's type (chat, embedding, responses, etc.).
  </Tip>
</Step>

<Step title="Monitor usage">
  Go to **Usage** in the sidebar to track your token consumption, request counts, and costs broken down by model and team.
</Step>
</Steps>

## Using the CLI

You can also authenticate and check your profile from the command line:

```bash
# Set up the CLI
cd cli && uv sync

# Configure your tenant and portal URL
uv run apollos configure

# Authenticate with device code flow
uv run apollos login

# Check your portal profile
uv run apollos me
```

See the [CLI overview](/cli/overview) for the full command reference.
